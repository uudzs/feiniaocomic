Debug 模式设置

说明：

• 开启 Debug 模式后，系统会显示详细的错误信息和调试信息

• 关闭 Debug 模式后，系统只显示友好的错误提示

• 生产环境建议关闭 Debug 模式，以提高系统安全性和性能

• 修改后会立即生效，无需重启服务

注意事项
  • 请确保 .env 文件具有写入权限（建议权限：644 或 664）
  • 如果无法修改，请检查服务器文件权限设置
  • 建议在修改前备份 .env 文件
  • Debug 模式开启时，系统可能会暴露敏感信息，请谨慎使用
